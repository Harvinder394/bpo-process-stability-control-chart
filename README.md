# 📊 BPO Process Stability & SLA Performance Analysis

## 🔍 Overview
This project analyzes the performance of a logistics offshore voice process operating across two locations (Hyderabad & Noida).

The objective was to evaluate whether the process is statistically stable and capable of meeting SLA targets across key KPIs:
- AHT (Average Handle Time)
- Quality Score
- CSAT (Customer Satisfaction)

---

## 🎯 Problem Statement
Despite appearing stable, the process showed consistent underperformance across all KPIs:
- AHT above SLA
- Quality below SLA
- CSAT below SLA

The goal was to identify root causes and recommend actionable improvements.

## 🔑 Key Insights
- Process is statistically stable but not capable
- Increasing call volume impacts performance
- Infrastructure issues increase AHT
- Calibration gaps affect Quality scores
- Delivery partner issues significantly impact CSAT

---

## 🚀 Recommendations
- Improve internet & system reliability
- Ensure 100% calibration compliance
- Strengthen delivery partner SLAs

---

## 📈 Expected Impact
- Reduction in AHT
- Improved Quality consistency
- Enhanced Customer Satisfaction
